# Numerical Methods in Python

This repository contains Python implementations of important numerical methods used to solve mathematical problems like finding roots of equations and solving systems of linear equations.

---

## 📌 Methods Included

### 🔹 Root Finding Methods

* Bisection Method
* Regula Falsi (False Position) Method
* Newton-Raphson Method

### 🔹 Linear System Methods

* Jacobi Method
* Gauss-Seidel Method

---

## ⚙️ Requirements

* Python 3.x
* NumPy (for Jacobi and Gauss-Seidel)

Install NumPy using:

```
pip install numpy
```

---

## ▶️ How to Run

1. Download or clone the repository
2. Open terminal in the project folder
3. Run any file:

```
python bisection.py
```

Similarly, you can run other files:

```
python regula_falsi.py
python newton_raphson.py
python jacobi.py
python gauss_seidel.py
```
